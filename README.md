# Claude Skills

Custom skills and prompts for Claude Code AI assistant.

## What are Skills?

Skills are reusable prompt templates that extend Claude Code's capabilities. They live in `.claude/skills/` directories and can be invoked with slash commands like `/skill-name`.

## Structure

```
skills/
├── skill-name/
│   └── SKILL.md          # Skill definition and instructions
└── another-skill/
    └── SKILL.md
```

## Usage

1. Copy a skill folder to your project's `.claude/skills/` directory
2. Invoke with `/<skill-name>` in Claude Code
